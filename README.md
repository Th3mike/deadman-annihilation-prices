# Deadman Annihilation Prices

Deadman Annihilation Prices is a RuneLite plugin designed to provide real-time economic data for the Deadman Mode: Annihilation season. By fetching live data directly from the official OSRS Wiki Price API, this plugin helps players make informed decisions on the Grand Exchange.

## Features

- **Live Wiki Integration**: Fetches the latest 'high' (buy) and 'low' (sell) prices specifically for the Deadman mode.
- **GE Inline Overlay**: Displays Wiki prices directly within the Grand Exchange offer setup screen, right below the Buy Limit.
- **Professional Search Panel**: A dedicated sidebar search tool featuring:
  - Real-time item filtering with OSRS-style icons.
  - Vertical price layout (Buy/Sell) for high readability.
  - GE Buy Limits displayed for every searchable item.
- **Authentic Aesthetic**: Fully integrated with the RuneLite interface using official RuneScape fonts and colors.

## Usage

1. **Overlay**: Simply click on an item to buy or sell it in the Grand Exchange. The "Wiki (Annihilation)" overlay will appear with the latest data.
2. **Search**: Click the **Crimson Skull** icon on the RuneLite sidebar to open the DMM Search panel. Type any item name to see its current market value and buy limit.

## Configuration

You can customize the plugin settings in the RuneLite configuration menu:
- **Show Wiki Buy Price**: Toggle the display of high prices.
- **Show Wiki Sell Price**: Toggle the display of low prices.
- **Refresh Interval**: Set how often the plugin updates its price cache (Default: 1 minute).
