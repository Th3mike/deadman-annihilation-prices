Próximos Passos Obrigatórios:

  • 1. Abra o IntelliJ IDEA no seu computador.
  • 2. Vá em File -> Open e selecione f:\GitHub\deadman-wiki-price.
  • 3. O IntelliJ detectará o pom.xml. Clique em "Load Maven Project".
  • 4. Se o erro persistir, no IntelliJ, abra o terminal (Alt+F12) e digite:
    mvn clean install -U
  • 5. Importante: Se o Maven ainda falhar, verifique se o IntelliJ está usando o repositório https://repo.runelite.net adicionado ao POM.
